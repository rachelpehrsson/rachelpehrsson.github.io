   Dynamic Quiz Distinguished Major Project - Wrote automated quiz system using HTML, CSS, PHP, and JavaScript that dynamically adds review material and extra questions depending on the student’s performance on quiz questions and use of hints.    Weather Tracking with Machine Learning - Java program to filter out noise and unwanted data from radar trackings of cloud patterns. It uses a neural network algorithm to analyze numerical readings within thousands of radar pulses, detect areas previously hand-marked as unwanted, and then use the generated machine to process new data files. Worked on as part of the NREIP internship at the Dahlgren Naval Warfare Center.
